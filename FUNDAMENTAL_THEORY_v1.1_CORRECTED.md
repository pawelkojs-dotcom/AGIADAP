## III. LEVEL 1: TWO-LINE LAW

### 3.1 Derivation from Level 0

**Coarse-graining procedure (Wilson, 1975; Goldenfeld, 1992):**

1. Start with quasiparticle-level description (Level 0)
2. Integrate out fast degrees of freedom
3. Obtain effective field theory

**Result:**

```
F_eff[σ; Θ] = -Θ · log ∫ exp(-F[σ, {s_i}]/Θ) Π_i ds_i
```

**First-order expansion yields:**

```
F[σ; Θ] = E[σ] - Θ·S[σ] + C[σ, E]
```

Where:
- **E[σ]:** Effective internal energy (renormalized)
- **S[σ]:** Effective entropy (renormalized)
- **C[σ, E]:** Coupling to environment (from D_ij integration)

### 3.2 Two-Line Law (Canonical Form)

**Line 1: Free Energy Functional**
```
F[σ; Θ] = E[σ] - Θ(x,t)·S[σ]
```

**Line 2: Dynamics Equation**
```
γ(x,t)·∂σ/∂t = -δF/δσ + √(2Θ(x,t))·ξ(x,t)
```

**Field Definitions:**

**σ(x,t) - Coherence Field**
- State of adaptonic system
- Structural ordering field
- **AGI:** belief/knowledge coherence
- **Universal:** stress field (Kojs, 2025)

**Θ(x,t) - Information Temperature**
- Exploration intensity
- Reorganization amplitude
- **AGI:** sampling temperature, novelty drive
- **Universal:** intensive reorganization parameter

**γ(x,t) - Medium Viscosity**
- Temporal adaptation metric
- Damping rate
- **AGI:** update damping, learning rate multiplier
- **Universal:** relaxation timescale

### 3.3 Physical Interpretation

**Energy Balance:**

```
dF/dt = -∫ γ·(δF/δσ)² dx + noise_terms
        \_____________/     \__________/
        Dissipation         Fluctuation
```

**Without noise (Θ=0):**
- Pure dissipation
- F decreases monotonically (Lyapunov function)
- System freezes at local minimum

**With noise (Θ>0):**
- Balance between dissipation and fluctuation
- F fluctuates around minimum
- System can escape local minima (activated processes)

**Fluctuation-Dissipation Theorem (Kubo, 1966):**

```
⟨ξ(x,t)·ξ(x',t')⟩ = δ(x-x')·δ(t-t')
Noise strength ∝ γ·Θ
```

Ensures thermalization to equilibrium (Boltzmann-like distribution):
```
P_eq(σ) ∝ exp(-F[σ]/Θ)
```

### A.1 D_ij Functional (Full Form)

**General expression:**

```
D_ij = ∫∫ K(x,y) · g(σ(x), σ(y), s_i(x), s_j(y)) dx dy
```

Where:
- **K(x,y):** Kernel function (e.g., Gaussian, exponential decay)
- **g(...):** Nonlinear coupling function (to be determined—open question)
- **Integration:** Over spatial/abstract domains Ω_i, Ω_j

**Quadratic approximation (used in current work):**

```
D_ij ≈ κ_ij · ∫ [σ_i(x) - σ_j(x)]² dx
```

**Gradient form (alternative):**

```
D_ij = λ_ij · ∫ [∇σ_i · ∇σ_j] dx + μ_ij · ∫ [σ_i · σ_j] dx
```

First term: gradient coupling (stiffness)  
Second term: direct coupling (alignment)

### A.2 Two-Line Law (Detailed Derivation)

**Variational derivative:**

```
δF/δσ(x) = ∂E/∂σ(x) - Θ(x)·∂S/∂σ(x)
```

**For E = ∫ [½(∇σ)² + V(σ)] dx (Ginzburg-Landau form):**

```
δE/δσ = -∇²σ + V'(σ)
```

**For S = -∫ σ·log(σ) dx (Shannon-like):**

```
δS/δσ = -(log(σ) + 1)
```

**Therefore, dynamics equation becomes:**

```
γ·∂σ/∂t = ∇²σ - V'(σ) + Θ·(log(σ) + 1) + √(2Θ)·ξ
```

This is the concrete realization of the abstract two-line law.

```python
class MultiLayerBaryon:
    """Multi-layer adaptonic system with explicit D_ij coupling"""
    
    def __init__(self, n_cores=3, dim=64):
        self.n_cores = n_cores
        self.cores = [AdaptonicCore(dim=dim) for _ in range(n_cores)]
        
        # Explicit interface layers (D_ij)
        self.interfaces = []
        for i in range(n_cores):
            for j in range(i+1, n_cores):
                self.interfaces.append(
                    InterfaceLayer(self.cores[i], self.cores[j])
                )
    
    def compute_D_ij(self):
        """Compute total interface coupling energy"""
        return sum(iface.coupling_energy() for iface in self.interfaces)
    
    def compute_F(self):
        """Total free energy: local + coupling"""
        F_local = sum(core.free_energy() for core in self.cores)
        F_coupling = self.compute_D_ij()
        return F_local + F_coupling
    
    def step(self, dt=0.01):
        """Single dynamics timestep (Langevin)"""
        for core in self.cores:
            # Local gradient
            grad_local = core.compute_gradient()
            
            # Coupling force from interfaces
            grad_coupling = sum(
                iface.coupling_gradient(core) 
                for iface in self.interfaces 
                if core in iface.cores
            )
            
            # Noise
            noise = np.sqrt(2 * core.Theta * dt) * np.random.randn(*core.shape)
            
            # Update (Langevin dynamics)
            core.sigma += dt * (-grad_local - grad_coupling) / core.gamma + noise
```

### C.2 Ca_e Controller (PI Implementation)

```python
class CaEController:
    """PI controller to maintain Ca_e ≈ 1.0"""
    
    def __init__(self, target=1.0, K_p=0.1, K_i=0.01):
        self.target = target
        self.K_p = K_p  # Proportional gain
        self.K_i = K_i  # Integral gain
        self.integral = 0.0
        
    def update(self, Ca_e_current, dt, weights):
        """
        Adjust λ weights to maintain Ca_e ≈ target
        
        Args:
            Ca_e_current: Current Ca_e value
            dt: Time step
            weights: Dict with keys 'epi', 'risk', 'coh'
        
        Returns:
            Updated weights dict
        """
        # Error
        error = self.target - Ca_e_current
        
        # Integral term (with anti-windup)
        self.integral += error * dt
        self.integral = np.clip(self.integral, -10, 10)
        
        # PI control signal
        adjustment = self.K_p * error + self.K_i * self.integral
        
        # Update weights (epistemic vs. risk trade-off)
        weights['epi'] *= (1 + adjustment)
        weights['risk'] *= (1 - adjustment)
        
        # Normalize to prevent drift
        total = sum(weights.values())
        weights = {k: v/total for k, v in weights.items()}
        
        return weights
```

### C.3 n_eff Calculation (Operational Definition)

```python
def compute_n_eff(states):
    """
    Compute effective number of channels via PCA-Shannon entropy
    
    Note: This is an operational measure analogous to participation 
    ratio in complex systems (Thouless, 1974), not standard physical 
    effective degrees of freedom.
    
    Args:
        states: Array of shape (n_timesteps, n_agents, state_dim)
    
    Returns:
        n_eff: Effective dimensionality (float)
    """
    from sklearn.decomposition import PCA
    
    # Flatten to (n_timesteps, n_agents * state_dim)
    n_timesteps, n_agents, state_dim = states.shape
    states_flat = states.reshape(n_timesteps, -1)
    
    # PCA
    pca = PCA()
    pca.fit(states_flat)
    
    # Eigenvalue distribution
    eigs = pca.explained_variance_
    p = eigs / eigs.sum()
    
    # Shannon entropy of eigenvalue distribution
    H = -sum(pi * np.log(pi) for pi in p if pi > 1e-10)
    
    # n_eff = exp(H)
    # Interpretation: "How many independent channels?"
    return np.exp(H)
```

**Important note:** This operational definition provides a useful empirical measure but differs from standard physical definitions of effective dimensionality. It should be validated against alternative metrics in future work.
