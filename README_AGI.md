# AGI Adaptonics — Starter Kit (σ–Θ–γ)

**Scope:** This repository bootstraps the **AGI thread** of Adaptonics as a **standalone project**, using the two‑line law and the three fundamental fields from Adaptonic Fundamentals 2.0.

```
F[σ; Θ] = E[σ] - Θ(x,t) · S[σ]
γ(x,t) · ∂t σ(x,t) = - δF/δσ(x,t) + √(2 Θ(x,t)) · ξ(x,t)
```
**σ** — coherence of beliefs/knowledge, **Θ** — exploration intensity, **γ** — medium viscosity (temporal metric).

## How to use this kit
1. Start every work session by pasting the content of `SESSION_BOOTSTRAP_AGI.md` into your assistant and updating the fields.
2. Treat `KERNEL_AGI.md` as **canonical** for AGI-domain definitions and equations.
3. Keep `CONCORDANCE_AGI.md` aligned with the universal canon and with other domains (cosmology, HTSC).
4. Log every change using `BACKLOG_AGI_CR.md` and record decisions with `ADR_AGI_TEMPLATE.md` (copy per decision).
5. Validate with `INVARIANTS_AGI.md` and evaluate with `EVAL_AGI.md` / `METRICS_AGI.md`.
6. Execute experiments following `EXPERIMENTS_AGI.md`.
7. Maintain interfaces in `INTERFACES_AGI.md`; add safety considerations in `SAFETY_AGI.md`.
