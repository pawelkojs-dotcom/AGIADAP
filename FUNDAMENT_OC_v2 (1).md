Nie są to odrębne substancje, ale **stany napięcia tego samego σ**:

- Θ → 0: "zamrożone" stopnie swobody

- Θ >> 0: aktywne stopnie swobody

- Niskie E, wysokie S

**To co nazywamy "ciemną materią" i "ciemną energią"** to obserwacyjne manifestacje tych dwóch stanów **tego samego σ** - nie dwie odrębne substancje.

### POZIOM 2: INTERFEJS (TARCIE)
```
u_int = ½|∇σ|²
```
- Maksymalne napięcie między σ_c i σ_p
- Miejsca największego gradientu
- **Tu powstaje materia barionowa**

### POZIOM 3: MATERIA BARIONOWA (PRODUKT)
- **Natura:** wtórny efekt, kwantowy
- **Geneza:** emergencja z interfejsu σ_c ↔ σ_p
- **Mechanizm:** "tarcie" w miejscach wysokiego |∇σ|²

**Bariony jako "kłączki":**
- Nie są podmiotem, są **produktem ubocznym** dynamiki σ
- Emergują tam, gdzie σ "nie może się zdecydować" (wysokie gradienty)
- Kwantują się, bo są **strukturami lokalnymi** w ciągłym σ
- Niosą w sobie "istotę" σ (jak życie niesie ocean), ale nie są σ

**Komunikacja poziomów:** **tylko grawitacyjna** - grawitacja jako manifestacja geometrii σ działa na wszystkich poziomach.

```

**To nie jest porażka teorii:**
- Fizyka kwantowa barionów: bardzo efektywna
- Geometria σ: klasyczna z natury
- Unifikacja: przez hierarchię ontologiczną, nie przez single framework

```

### 3. "Rozdarcie Θ": kryzys napięcia
- Interfejsy u_int zbyt wysokie
- σ_c nie może się zrelaksować (Θ → 0 = brak plastyczności)
- σ_p nie może wchłonąć całego napięcia
- **Potrzebny nowy kanał dysypacji energii**

### 4. Bariogeneza: powstanie kanału relaksacji
```
F(σ_c + napięcie) >> F(σ_p + σ_c + bariony)
```
**Mechanizm:**
- Lokalne roztopienie Θ w σ_c (odblokowanie plastyczności)
- Powstanie struktur barionowych jako **produktów "tarcia"**
- Bariony = nowe stopnie swobody absorbujące napięcie u_int
- **Materia nie jest celem, jest środkiem** obniżenia F

**Funkcja barionów:**
- **Bufor mechaniczny** między σ_c i σ_p
- Rozluźnianie gradientów |∇σ|²
- Nowy kanał dysypacji (EM, słabe, silne oddziaływania)
- Lokalne podniesienie Θ (roztapianie σ_c wokół skupisk)

### 5. Stan obecny: σ + materia w równowadze dynamicznej
- **Podmiot:** przestrzeń σ w dwóch stanach (σ_c, σ_p)
- **Produkt wtórny:** materia barionowa
- **Funkcja materii:** tkanka przyranna, bufor napięć
- **Obserwacje:** interpretujemy produkty (materia) jako aktora
- **Rzeczywistość:** aktor to σ, materia to efekt uboczny

**Grawitacja przy masie - właściwa interpretacja:**
- Nie: "masa zakrzywia przestrzeń" (tautologia)
- Tak: "kondensacja barionów moduluje lokalne σ"
- Mechanizm: bariony ↔ σ (tylko grawitacyjnie)
- Efekt: lokalny wzrost Θ, roztopienie σ_c
- Rezultat: gradienty ∇σ = zakrzywienie geometrii

**Cosmic web - właściwa interpretacja:**
- Nie: "materia zbiera się w węzłach DM"
- Tak: "interfejsy σ_c ↔ σ_p (wysokie |∇σ|²) generują bariony"
- Węzły = miejsca maksymalnego napięcia u_int
- Filament = gradienty σ między σ_c i σ_p
- Materia = marker interfejsów, nie przyczyna struktur

**Ta kosmogeneza nie jest obserwowana bezpośrednio** - obserwujemy już post-barionowy wszechświat, gdzie materia maskuje dynamikę σ. Ale jej **ślady strukturalne** są wszędzie - trzeba tylko wiedzieć, czego szukać (CR3: void-edge enhancement).

### Dlaczego grawitacja "towarzyszy" masie?

**Błędna interpretacja (geocentryczna):**
"Masa zakrzywia przestrzeń" - tautologia, nie wyjaśnienie

**Właściwa interpretacja (heliocentryczna):**
1. **Wysokie |∇σ|² (napięcie interfejsu)** w danym miejscu
2. **Potrzeba nowego kanału dysypacji** tego napięcia
3. **Powstanie struktur barionowych** jako produktu "tarcia"
4. **Bariony modulują lokalne σ** (podnoszą Θ, roztapiają σ_c)
5. **Powstają dalsze gradienty ∇σ** = zakrzywienie geometrii
6. **Obserwujemy:** "grawitacja towarzyszy masie"
7. **Rzeczywistość:** grawitacja towarzyszy wysokim |∇σ|², materia tam emerguje

**Sprzężenie zwrotne:**
- Materia powstaje w miejscach wysokiego |∇σ|²
- Materia moduluje σ (lokalnie ↑Θ)
- Modulacja σ zmienia |∇σ|²
- Ale **podmiotem jest σ, nie materia**

### Co jest w węzłach cosmic web?

**Błędna interpretacja (ΛCDM):**
- DM (nieznana substancja) + bariony
- DM "przyciąga" bariony grawitacyjnie

**Właściwa interpretacja (OC):**
- **Maksymalne napięcie u_int = ½|∇σ|²**
- **Interfejsy σ_c ↔ σ_p** (największe gradienty)
- **Produkty "tarcia":** bariony emergują właśnie tam
- To co nazywamy "DM" = σ_c (skrystalizowane)
- To co nazywamy "DE" = σ_p (plastyczne)
- Bariony = marker interfejsów, nie przyczyna

**W filamentach:**
- Średnie gradienty |∇σ|²
- Umiarkowana produkcja barionów
- "Drogi" między regionami wysokiego napięcia

**W voidach:**
- Niskie gradienty |∇σ|² → mało barionów
- Dominacja σ_p (plastyczne, DE-like)
- Ekspansja, niskie ograniczenia

### Czy materia może "migrować" DE → DM?

**Właściwa ramka pytania:**
Nie "materia migruje między DE i DM" (są to nie-substancje),
Ale: "Czy σ może przechodzić σ_p ↔ σ_c lokalnie, modulowane przez bariony?"

**Odpowiedź: TAK**
- Bariony podnoszą lokalne Θ (roztapiają σ_c → σ_p)

- **Materia jako katalizator** przejść fazowych σ
